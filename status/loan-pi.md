## 주요 현황

| 항목 | 상태 | 비고 |
|------|:----:|------|
| PoC 시뮬레이터 | ✅ | 터치 기반 UX, 시나리오 분기 + LLM 슬롯 추출 |
| 디금센 현장 조사 | ✅ | 인력 구조(22+2+6), 페인포인트 파악 완료 |
| UI 방식 결정 | ✅ | 앞단 버튼식(불가 필터링) + 뒷단 LLM 대화형 |
| 디금센 현장 방문 (역삼) | ✅ | 4/8 방문 완료 (PoC 시연 + 피드백 수집) |
| 비대면 프로세스 분석 | ✅ | SOL 12단계 분석, 경쟁사 비교(카카오/토스/국민) 완료 |
| 임대차 계약서 OCR PoC | 🔄 | 중개대상물 확인설명서 교차검증 설계 진행중 |
| MCP 툴 개발 요청 | 🔄 | 여신 서비스 개발부 협의 필요 (신용평가/보증한도 조회) |
| 비대면 플로우 작성 | ⏳ | 전성민 팀장, 슈퍼SOL 기준 |
| 파일럿 점포 선정 | ⏳ | 스마트혁신점포 1~2곳 (대출존 활용도 높은 곳) |
| 봇프레임워크 운영 | ⚠️ | 저작자 퇴사/연락두절, AICC 기반 구축 불가 → 별도 구현 |
